# Dataset — тестовые документы

Набор текстовых файлов для разработки и проверки функций `extract()`, `classify()` и `check_subject()`.

## Файлы

| Файл | Тип | Особенности |
|---|---|---|
| `contract_001.txt` | contract | Стандартный договор поставки. Сумма прописью + цифрами, дата в формате `01.03.2025` |
| `spec_001.txt` | spec | Спецификация с таблицей. Сумма в формате `1 250 000,00 руб.` |
| `invoice_001.txt` | invoice | Счёт с несколькими форматами суммы: `1 250 000 руб.`, `1,250,000.00 RUB` и прописью |
| `invoice_002.txt` | invoice | Счёт с суммой только прописью, дата в формате `28/02/25` |
| `act_001.txt` | act | УПД с таблицей, дата `24 марта 2025 г.` |
| `act_002.txt` | act | Акт выполненных работ, сумма в формате `500000.00 ₽`, НДС не облагается |
| `scan_ocr_001.txt` | unknown | Имитация OCR-мусора: буквы перепутаны с цифрами (O→0, l→1), тип определить сложно |
| `subjects_test.txt` | — | 15 предметов оплаты с ожидаемым результатом для `check_subject()` |
